
# ASAPI

AgroSpace API R Packages

More than 20 satellite scientific index to be use in your application. 

Full documentation in [api.agrospace.cl](api.agrospace.cl)




```
# install ASAPI

remotes::install_github('agrospace/ASAPI')

```
